# 用下载的代码新建keil工程

## 新建存放keil工程的文件夹

```shell
$ mkdir -p code/Project/MDK-ARM
$ tree
.
└── code
    └── Project
        └── MDK-ARM
```

新建工程时，工程文件存放在该目录下。
